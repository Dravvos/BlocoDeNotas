# 📝 Bloco de Notas — Windows Forms (C#)

Um aplicativo desktop de bloco de notas desenvolvido em **C# com Windows Forms**, inspirado no Bloco de Notas nativo do Windows. Permite criar, editar, formatar e salvar arquivos de texto de forma simples e intuitiva.

---

## ✨ Funcionalidades

- **Criar, abrir e salvar** arquivos de texto (`.txt`)
- **Formatação básica**: negrito, itálico e sublinhado
- **Desfazer e refazer** alterações (Undo/Redo)
- **Copiar, colar e recortar** texto
- Interface limpa e familiar, semelhante ao Bloco de Notas do Windows

## 🚀 Como Executar

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/Dravvos/BlocoDeNotas.git
   ```

2. **Abra o projeto no Visual Studio:**
   - Acesse `Arquivo > Abrir > Projeto/Solução`
   - Navegue até a pasta clonada e selecione o arquivo `.sln`

3. **Compile e execute:**
   - Pressione `F5` ou clique em **Iniciar** para rodar o projeto

4. **Utilizando o aplicativo:**
   - Use as opções do menu para abrir, salvar ou formatar arquivos
   - Salve suas notas como arquivos `.txt` em qualquer local do computador
